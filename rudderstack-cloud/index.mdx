# RudderStack Cloud

RudderStack lets you collect, transform, and route customer data across your entire stack via efficient event streaming, ETL, and reverse-ETL pipelines.

**The easiest way to get started with RudderStack is by [signing up](https://app.rudderstack.com/signup) for RudderStack Cloud**.

This section contains useful guides on setting efficient, real-time data pipelines in RudderStack.
