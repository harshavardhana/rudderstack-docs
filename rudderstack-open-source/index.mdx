# RudderStack Open Source

The easiest way to get started with RudderStack is to [**sign up**](https://app.rudderstack.com/signup) for [**RudderStack Cloud**](https://rudderstack.com/docs/rudderstack-cloud/). 

The guides in this section are useful if you want to set up open-source RudderStack in your preferred development environment.

<div class="infoBlock">

The <a href="https://github.com/rudderlabs/rudder-server">RudderStack backend</a> is written in Go, with a simple React.js UI to manage your sources, destinations, and the connections between them.
</div>
