# Home

[**RudderStack**](https://rudderstack.com/) is a **customer data platform for developers.** Our tooling makes it easy to deploy pipelines that collect customer data from every app, website and SaaS platform, then activate it in your warehouse and business tools.

<div class="successBlock">

Start building a better, warehouse-first CDP that delivers complete, unified data to every part of your customer data stack. <a href="https://app.rudderstack.com/signup">Sign up</a> for <a href="https://app.rudderstack.com/signup">RudderStack</a> today.
</div>

## Quick start

- Sign up for [**RudderStack**](https://app.rudderstack.com/signup).
- Start sending data within 5 minutes using RudderStack. [**Get Started**](https://rudderstack.com/docs/get-started/).
- Learn how RudderStack tracking works. Send `track`, `identify`, and other types of events via RudderStack. Refer to the [**RudderStack Events Specification**](https://rudderstack.com/docs/rudderstack-api/api-specification/rudderstack-spec/).
- Stuck somewhere? Read the [**user guides**](https://rudderstack.com/docs/user-guides/) or check out the [**FAQ section**](https://rudderstack.com/docs/faqs/).

<div class="successBlock">
    
For community support, start a conversation in our <a href="https://rudderstack.com/join-rudderstack-slack-community">Slack community</a>.
</div>
